# Gestión de Proyectos – Proyecto Musical

Este repositorio reúne casos reales de gestión de proyectos en el ámbito cultural y musical, con foco en planificación, coordinación de equipos, comunicación institucional y ejecución en contextos reales.

Los proyectos acá documentados reflejan mi experiencia en gestión de personas, organización de procesos y toma de decisiones bajo presión, con impacto medible y entregables concretos.

## Contenido

- Producción y gestión integral de shows
- Coordinación de equipos técnicos y artísticos
- Comunicación institucional y difusión
- Planificación, ejecución y evaluación de proyectos

Este repositorio forma parte de mi portfolio profesional.
